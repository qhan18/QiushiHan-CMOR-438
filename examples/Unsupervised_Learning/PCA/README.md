# Principal Component Analysis (PCA)

## Overview
Demonstrates **PCA** implemented from scratch using eigendecomposition of the covariance matrix for dimensionality reduction.

## Dataset
**Digits Dataset** (sklearn) — 1797 handwritten digit images, 64 features (8x8 pixels), 10 classes.

## Results
| Goal | Components Required |
|---|---|
| 95% variance retained | 40 |
| 99% variance retained | 54 |
| 2D visualization | 2 (21.59% variance) |

## Key Concepts
- Eigendecomposition, covariance matrix, principal components, explained variance ratio, dimensionality reduction, reconstruction
