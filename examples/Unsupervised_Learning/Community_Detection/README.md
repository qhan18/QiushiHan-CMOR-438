# Label Propagation (Community Detection)

## Overview
Demonstrates **Label Propagation** implemented from scratch using RBF kernel similarity for semi-supervised learning and community detection.

## Datasets
- Blob dataset with 10% labeled points for semi-supervised classification
- Moons dataset to demonstrate non-linear community detection

## Results
| Dataset | Labeled Points | Accuracy |
|---|---|---|
| Blobs | 20 of 200 (10%) | varies by fraction |
| Moons | 40 of 200 (20%) | 87.50% |

## Key Concepts
- RBF kernel, similarity graph, semi-supervised learning, label spreading, gamma sensitivity, community detection
