# DBSCAN Clustering

## Overview
Demonstrates **DBSCAN** implemented from scratch, highlighting its ability to find arbitrarily shaped clusters and detect noise points automatically.

## Datasets
- Moons dataset for non-linear cluster demonstration
- Blobs with intentional noise for noise detection demonstration

## Results
| Dataset | Clusters Found | Noise Points |
|---|---|---|
| Moons | 2 (correct) | 0 |
| Noisy Blobs | 3 (correct) | 7 of 30 added |

## Key Concepts
- Core points, border points, noise points, eps sensitivity, density-based clustering, arbitrary cluster shapes
