# K-Means Clustering

## Overview
Demonstrates **K-Means Clustering** implemented from scratch using Lloyd's algorithm with random centroid initialization.

## Dataset
Synthetic blob dataset generated with make_blobs, 300 samples, 4 true clusters.

## Results
| Metric | Value |
|---|---|
| Optimal k | 4 |
| Inertia at k=4 | 11.3170 |
| Cluster sizes | 75, 75, 75, 75 |

## Key Concepts
- Centroid initialization, iterative assignment, inertia, elbow method, cluster convergence
