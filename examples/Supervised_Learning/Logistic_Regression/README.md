# Logistic Regression

## Overview
Demonstrates **Logistic Regression** implemented from scratch using gradient descent to minimize binary cross-entropy loss.

## Dataset
**Breast Cancer Wisconsin Dataset** (sklearn) — 569 samples, 30 features, binary target (malignant vs benign).

## Results
| Metric | Score |
|---|---|
| Accuracy | 98.2% |
| Misclassifications | 2/114 |

## Key Concepts
- Sigmoid function, binary cross-entropy, gradient descent, decision boundary
