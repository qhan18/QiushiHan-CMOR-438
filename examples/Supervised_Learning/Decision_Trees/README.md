# Decision Tree Classifier

## Overview
Demonstrates a **Decision Tree Classifier** implemented from scratch using entropy-based information gain for split selection.

## Dataset
**Iris Dataset** (sklearn) — 150 samples, 4 features, 3 classes.

## Results
| Max Depth | Train Accuracy | Test Accuracy |
|---|---|---|
| 1 | 66.67% | 66.67% |
| 2 | 96.67% | 93.33% |
| 3 | 98.33% | 96.67% |
| 5+ | 100.00% | 93.33% |

## Key Concepts
- Entropy, information gain, recursive splitting, axis-aligned boundaries, overfitting, max depth as regularization
