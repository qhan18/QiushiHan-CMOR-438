# K-Nearest Neighbors (KNN)

## Overview
Demonstrates **KNN** implemented from scratch using Euclidean distance and majority voting for classification.

## Dataset
**Iris Dataset** (sklearn) — 150 samples, 4 features, 3 classes (setosa, versicolor, virginica).

## Results
| k | Accuracy |
|---|---|
| 1 | 96.7% |
| 3 | 93.3% |
| 5 | 93.3% |
| 7-21 | 96.7% |

## Key Concepts
- Euclidean distance, majority voting, decision boundaries, bias-variance tradeoff in k selection
