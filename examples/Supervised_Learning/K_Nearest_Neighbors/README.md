# K-Nearest Neighbors (KNN)

## Overview
Demonstrates **KNN** implemented from scratch using Euclidean distance and majority voting for classification.

## Dataset
**Wine Dataset** (sklearn) — 178 samples, 13 chemical features, 3 wine cultivar classes.

## Results
| k | Accuracy |
|---|---|
| 1 | 97.22% |
| 3 | 97.22% |
| 5 | 97.22% |
| 7-21 | 100.00% |

## Key Concepts
- Euclidean distance, majority voting, decision boundaries, bias-variance tradeoff in k selection, feature scaling
